Encoder-Decoder Pipeline

Understanding how computer vision and NLP are fused together.

Feature Extractors (Encoders)

ResNet50 (Residual Network)

Uses deep residual skip-connections to allow gradients to flow directly through layers. We extract features from the pool5 layer (outputs a 2048-dim feature map), then run it through a trained Linear layer to map to 256 dimensions.

Highly Recommended

VGG16 (Visual Geometry Group)

A classic deep CNN characterized by small 3x3 convolution filters stacked sequentially. We run the output of its conv5 block through a spatial adaptive average pooling layer to collapse 3D features to 512 dimensions, which are projected to 256 dimensions.

Classic Alternative

Language Decoders

DecoderRNN (LSTM Model)

Uses a Long Short-Term Memory cell. In this configuration, the image feature is prepended to the sequence of word tokens as Step 0. The hidden state propagates context sequentially word-by-word. Highly effective for simple grammatical captions.

Sequential Context

DecoderTransformer

A modern self-attention based model. The image feature and the caption tokens are processed as a single sequence. We use a causal look-ahead mask to ensure the model cannot peek at future words during training. Allows parallel self-attention computation.
